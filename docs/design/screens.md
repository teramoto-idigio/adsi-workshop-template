# 画面設計

## 画面一覧

| # | 画面名 | パス | ロール | 説明 |
|---|--------|------|--------|------|
| 1 | ログイン | /login | 全員 | メール＋パスワード入力 |

| 7 | 承認一覧 | /approvals | マネージャー | 有給/残業の未承認一覧 |
| 8 | 部署メンバー勤怠 | /team | マネージャー | メンバーサマリー一覧 |

## ワイヤーフレーム

### 共通レイアウト

```
┌─────────────────────────────────────────────┐
│ [Logo] 勤怠管理      [User Name ▼] [Logout] │
├────────┬────────────────────────────────────┤
│ Nav    │                                    │
│        │                                    │
│ 勤怠   │          Main Content              │
│ 有給   │                                    │
│ 残業   │                                    │
│ ---    │                                    │
│ 承認   │                                    │
│ チーム  │                                    │
│ ---    │                                    │
│ 社員   │                                    │
│ 部署   │                                    │
│ 有給付与│                                    │
└────────┴────────────────────────────────────┘
```

- Nav はロールに応じて表示項目が変わる
- 社員: 勤怠/有給/残業
- マネージャー: + 承認/チーム
- 管理者: + 社員/部署/有給付与

## コンポーネント一覧

| コンポーネント | 用途 |
|--------------|------|
| AppLayout | 共通レイアウト（Nav + Main） |
| Sidebar | サイドナビゲーション（ロール対応） |
| CalendarGrid | 月次カレンダーグリッド |
| CalendarCell | カレンダーの1日セル |
| AttendanceForm | 勤怠入力フォーム（モーダル内） |
| SummaryCard | 月次集計サマリーカード |
| LeaveRequestForm | 有給申請フォーム |
| OvertimeRequestForm | 残業申請フォーム |
| ApprovalList | 承認待ち一覧テーブル |
| EmployeeTable | 社員一覧テーブル |
| DepartmentTable | 部署一覧テーブル |
| ConfirmDialog | 確認ダイアログ（承認/却下/削除） |
